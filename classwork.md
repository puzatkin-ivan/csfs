# Классная работа

Для написания программ используем редактор `Kate`, который можно найти через `Пуск` во вкладке `Офис`
Для запуска программ используем `Терминал`, который можно найти через `Пуск` во вкладке `Системные`

**План действий**
1. Открываем редактор `Kate`
2. Читаем задание
3. Пишем программу в редакторе `Kate`
4. Сохраняем файл с названием `task<номер задания>.py`, где `<номер задания>` - порядковый номер задания(Например, для Задания №1 название должно быть `task1.py`)
5. Открываем `Терминал Fly`
6. Запускаем программу с помощью **python3** (Например, для Задания №1 запуск будет выглядеть как `python3 task1.py`)
7. Проверяем работу программы
8. В случае, если программа удовлетворяет требованиям, переходим к следующему заданию

### Задания №1(2 балла)
Дана програма, которая выводит на экран текст `How were the holidays?`

```python3

```


### Задание №3 (5 баллов)
Дана програма, которая выводит на экран текст `You need to say 'Bazinga!' and you will complete the task. It will work once!`

```python3
print("You need to say 'Bazinga!' and you will complete the task. It will work once!")
```
**Обратите внимание, что в тексте есть одинарные кавычки, поэтому текст в начале и в конце выделяется двойными кавычками**

Напишите программу и измените программу так, чтобы программа выводила на экране следующий текст:
```
You need to say 'Bazinga!' and you will complete the task. It will work once!
Next time, I will have better luck.
```

