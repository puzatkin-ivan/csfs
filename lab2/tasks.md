**Для AstraLinux**
Для написания программ используем редактор `Kate`, который можно найти через `Пуск` во вкладке `Офис`
Для запуска программ используем `Терминал`, который можно найти через `Пуск` во вкладке `Системные`

**План действий**
1. Открываем редактор `Kate`
2. Читаем задание
3. Пишем программу в редакторе `Kate`
4. Сохраняем файл с названием `task<номер задания>.py`, где `<номер задания>` - порядковый номер задания(Например, для Задания №1 название должно быть `task1.py`)

6. Запускаем программу с помощью **python3** (Например, для Задания №1 запуск будет выглядеть как `python3 task1.py`)
7. Проверяем работу программы
8. В случае, если программа удовлетворяет требованиям, переходим к следующему заданию

### Задания №4(5 балла)
Дана программа, которая получает от пользователя температуру в Цельсиях и переводит эту температуру во Фарангейты
Примечание: температура в цельсиях перводится в температуру в Фарангейты по формуле (Tф = (Tц * 9/5) + 32)
Добавьте в программу, чтобы она могла переводить температуру в Келвины
Температура в цельсиях перводится в температуру в келвины Tк = Tц + 273

```python3
celsius = int(input('Temperature in degree Celsius: '))
fahrenheit = (celsius * (9/5)) + 32
print('degree Fahrenheit:', fahrenheit)
```
p.s Тот, кто объяснит почему такая замудренная формула перевода в Фарангейты, тому +3 балла
