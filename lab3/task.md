# Классная работа

### Тема урока: Импорты, функции и массивы

Импорт - это подключение библиотеки с кодом к вашей программе, которые хранятся в других файлах.
Это позволяет использовать код, написанный нами и другими разработчиками в нашей программе.
Для того, чтобы создать файл, который можно будет подключать как библиотеку к нашей программе и через пробел написать название файла. Для того, чтобы подключить библиотеку к программе, нужно указать вначале файла ключевое слово `import`
Например, подключить файл `random.py` к программе нужно вначале программы указать `import random`
Для того, чтобы использовать функции из подключенной библиотеку перед функцией указывать название библиотеку, из которой вы хотите получить эту функцию.

**Задание №1**
`import time`
С помощью функции `time.sleep(1)` написать программу, которая выводит на экран обратный счет от заданного пользователем числа
Примечание: функция `time.sleep(1)` где 1 это кол-во секунд, которые программа бездействует

**Задания #2**
`import random`
Напишите программу, которая случайным образом выдает либо `Heads`, либо `Tails`
Для этого используйте массив значений `variants = ['Heads', 'Tails']`
Примечание: используйте функцию `random.choice`


**Задание №3**

Написать программу, которая получает от пользователя число и с помощью функции `math.sqrt` находит квадратный корень от польховательского числа

**Задание №4**
`import random`
Написать программу, которая создает 3 случайных числа и выводит на экран наибольше
Примечание: используйте функцию `random.randint(1, 100)` функция принимает 2 числа(начало диапозона, конец диапозона)

**Задание №5**
Петя зашёл в волшебный магазин сладостей. Там продают конфеты, пирожные и шоколадки, но касса принимает только чеканные монеты четырёх видов:  1 рубль  
5 рублей  
10 рублей  
25 рублей

Продавец очень занятой и не любит, когда ему дают кучу мелочи. Он просит Петю заплатить так, чтобы монет было как можно меньше.

Напиши программу, которая помогает Пете отдать ровно нужную сумму минимальным количеством монет.
**Ввод**: Одно целое число от 1 до 1000 — сколько рублей стоит покупка.
**Вывод** Одно число — минимальное количество монет, которыми можно расплатиться.
