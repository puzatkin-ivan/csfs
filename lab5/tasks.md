# Интерфейс командной строки

### Задание №1
1. Запустить `ls` через командную строку, команда должна вывести каталог папок(директорий) и файлов в текущей директории
2. Запустить `ls -lh`
3. Написать в тетради различие выдачи первого вызова команды `ls` от второго `ls -lh`

### Задание №2 (Создание папки/директории)
1. Запустить команду `mkdir` с названием класса(букву использовать латинскую a или b) и вашей группы через тире. Например, если учитесь в 7б и приходите вторым урок, то должны написать `mkdir 7b-2`
2. С помощью команды `ls` проверить, что папка(директории) с вашим названием создалась
3. С помощью команды `cd` перейти в директорию, которую вы создали
4. Выполнить команду `ls` результат выполнения записать в тетрадь
5. Выполните команду `cd ..` (выйти из директории)

### Задание №3
Запустить редактор Kate через командную строку
С помощью редактора Kate создать файл с название `numbers.txt` и сохранить его в директории, которую создали в **Задание №2**
В тетради запишите команды, которыми я смогу проверить, что файл создан

### Задание №4
Запустить редактор Kate через командную строку
Создать файл `hello.py`, в котором должна быть инструкция `print('Hello')` и сохранить его в директории, которую создали в **Задание №2**
В тетради написать как проверить, что файл создан
Запустите файл с помощью `python3`

### Задание №5
1. Зайти в директорию, которую вы создали
2. Проверить, что в ней 2 файла
3. Если файлов нет, то сначала выполните предыдущие задания
4. Выполните `cat numbers.txt`
5. Выполните `cat hello.py`
6. Запишите в тетрадь, что делает команда `cat`
