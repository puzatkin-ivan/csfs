### Задание по функции `print()`
Для вывода данных на экран используется функция `print()`.

Внутри круглых скобок пишем, что хотим вывести на экран. Если это текст, то обязательно указываем его внутри кавычек. Кавычки могут быть одинарными (`'`) или двойными (`"`).
> **До и после текста мы ставим только одинаковые кавычки**.
<br>

**Пример задания №1:**
Написать программу, которая выводит на экран `Hello from Sanchursk!`
Что нужно сделать:
Написать программу, которая вызывает функцию `print` и передает в эту функцию строку `Hello from Sanchursk!`
```python
print('Hello from Sanchursk!')
```

**Пример задания №2:**
Написать программу, которая на экран выводит заданную фигуру:
```
  *
 * *
*   *
 * *
  *
```
Программа:
```python
print('  *  ')
print(' * * ')
print('*   *')
print(' * * ')
print('  *  ')
```

### Задания по функции `input()`
Для считывания данных в языке Python используется функция `input()`.
В функцию `input()` можно передать строку `input('What is your name?')`.
Передавая строку `What is your name?` в функцию `input()`, мы говорим программе, чтобы она сначала вывела на экран текст, а потом прочитала данные, которые пользователь наберет с клавиатуры

**Пример задания№3:**
Написать программу, которая ожидает прочитает то, что пользователь наберет с клавиатуры имя `John` и выведет на экран `Hello John`
```pythonъ
name = input()
print('Hello', name)
```
При каждом запуске имя может меняться, значит, результат тоже должен меняться

**Пример задания №4:**
Написать программу, которая прочитает от пользователя 2 числа и выведет сумму 2 чисел
```python
number1 = int(input())
number2 = int(input())
print(number1, number2)
```
В проверочной работе могут быть и другие математические операции

>Функция input всегда читать только строки
>Функция `int()` позволяет преобразовать строку в число
>

### Задания по условному оператору
Условный оператор - конструкция в программировании, которая позволяет менять ход выполения в зависимости от того, выполняет проверочное условие или нет
Пользователь вводит число, если число четное, то нужно ввести `even`, если не четное, то `odd`
```python
number = int(input())
if number % 2 == 0:
  print('even')
else:
  print('odd')
```

Операция `%` - остаток от деления

**Пример задания №5:**
Написать программу, которая получает от пользователя 2 числа и выводит сначала наименьшее, а потом наибольшее число
```python
number1 = int(input())
number2 = int(input())
if number1 < number2:
  print(number1, number2)
else:
  print(number2, number1)
```

**Пример задания №6**
Написать программу, в которую требуется ввести пароль дважды. Если пароли совпадают, то вывести `confirmed`, иначе `no confirmed`
```python
password1 = input()
password2 = input()
if password1 == password2:
```
